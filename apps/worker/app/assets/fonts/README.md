# Bundled render font

`Inter-Variable.ttf` is the variable Inter font from the Google Fonts
repository. It contains the regular through ExtraBold weight range used by the
four ASS caption presets. FFmpeg receives this directory through libass'
`fontsdir` option, so render output does not depend on fonts installed on the
user's machine.
